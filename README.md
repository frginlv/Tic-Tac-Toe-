# This repo conataines a Tic Tac Toe game that is made with Python + Tkinter , with the help of some tutos on youtube (video on the channel : Kenny Yip Coding).

----------------

## How to install and play the game :

### On Linux or MacOs : (Run these command in the terminal)

- git clone https://github.com/frginlv/Tic-Tac-Toe.git
- cd Tic-Tac-Toe
- python3 main.py

### On Windows : (Run these command in the terminal)

- git clone https://github.com/frginlv/Tic-Tac-Toe.git
- cd Tic-Tac-Toe
- python main.py
